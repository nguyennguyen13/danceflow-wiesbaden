# DanceFlow Wiesbaden 

Willkommen im Repository von **DanceFlow Wiesbaden** – der Webanwendung für unsere Tanzschule in Wiesbaden. Das Projekt basiert auf Node.js, Express und EJS.

---

## Setup & Installationsanweisungen

Befolge diese Schritte, um das Projekt lokal auf deinem Rechner aufzusetzen:

### 1. Repository klonen und Verzeichnis wechseln
Öffne dein Terminal und führe folgende Befehle aus:
```bash
git clone [https://github.com/danceflow-wiesbaden/danceflow-wiesbaden.git](https://github.com/danceflow-wiesbaden/danceflow-wiesbaden.git)
cd danceflow-wiesbaden
